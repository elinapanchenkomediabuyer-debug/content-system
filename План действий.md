# ПОЛНЫЙ ПЛАН СОЗДАНИЯ КОНТЕНТ-СИСТЕМЫ ДЛЯ СОЦИАЛЬНЫХ СЕТЕЙ КЛИНИКИ

## Цель проекта

Создать единую систему, которая позволит быстро и качественно производить маркетинговые материалы для клиники:

- сценарии роликов;
- посты;
- флаеры;
- баннеры;
- материалы для экранов;
- ответы администраторам;
- тексты для рекламы;
- любые другие материалы по каждой услуге.

---

## ЭТАП 1. Определение стратегии продвижения

**1. Определить пулы услуг, которые будут продвигаться.**

Составить полный список услуг и аппаратов.

Распределить их по приоритету согласно:

- плану продаж;
- действующим акциям;
- стратегическим задачам клиники.

Именно эти пулы услуг станут основой всей дальнейшей системы.

**2. Определить количество роликов.**

Распределить 45 роликов между пулами услуг.

Затем распределить их между аккаунтами:

- 30 роликов — аккаунт косметологии.
- 15 роликов — аккаунт лечения.

**Результат этапа:**

- список пулов услуг;
- список аппаратов;
- количество роликов по каждой услуге;
- распределение между аккаунтами.

---

## ЭТАП 2. Создание базы знаний

Для каждого пула услуг выполнить полный цикл.

### 2.1. Собрать сырой материал.

Собрать максимально возможное количество информации:

- страницы сайта;
- описания услуг;
- описания аппаратов;
- инструкции;
- методические материалы;
- акции;
- FAQ;
- отзывы;
- кейсы;
- фотографии;
- посты Telegram;
- статьи;
- внутренние документы;
- любые дополнительные материалы.

**Важно:**

Информация собирается сразу по целой услуге, а не по отдельным темам.

### 2.2. Создать единый шаблон документа по услуге.

Утвердить структуру документа.

Например:

- описание услуги;
- для кого подходит;
- преимущества;
- боли клиента;
- страхи;
- возражения;
- ожидания;
- результаты;
- противопоказания;
- акции;
- ответы на вопросы;
- кейсы;
- фотографии;
- полезные факты;
- дополнительные материалы.

После утверждения шаблон больше не менять.

### 2.3. Автоматизировать оформление.

Пока собирается информация по следующей услуге, Claude оформляет предыдущую.

Получается конвейер.

В результате появляется единая база знаний по каждой услуге.

---

## ЭТАП 3. Методика Яны Ащиной

Подготовить два отдельных документа.

1. Косметология.
2. Лечение.

В каждом собрать максимально подробно:

- позиционирование;
- стиль коммуникации;
- структура роликов;
- способы удержания внимания;
- работа с болями;
- экспертность;
- продажи;
- призывы к действию;
- особенности медицинского копирайтинга;
- рекомендации по построению сценариев.

Эти документы становятся главными методическими материалами.

---

## ЭТАП 4. Создание идеальных шаблонов

Один раз создать идеальные шаблоны.

### 4.1. Шаблон документа по услуге.

### 4.2. Шаблон сценария.

Он должен содержать:

- хронометраж;
- этап сценария;
- цель этапа;
- через какую эмоцию идем;
- какую боль раскрываем;
- рекомендации Яны Ащиной;
- текст врача;
- текст модели;
- что происходит в кадре;
- действия врача;
- действия модели;
- монтаж;
- подписи;
- выделения;
- фотографии;
- инфографику;
- кейсы;
- дополнительные вставки;
- текст поста (при необходимости).

### 4.3. Шаблон портрета целевой аудитории.

### 4.4. Шаблон поиска тем.

### 4.5. Шаблон флаера.

### 4.6. Шаблон технического задания дизайнеру.

Все дальнейшие материалы должны строиться только на этих шаблонах.

---

## ЭТАП 5. Создание портретов целевой аудитории

Для каждого пула услуг сформировать отдельный документ.

Использовать:

- документ по услуге;
- методику Яны Ащиной.

Для каждой аудитории описать:

- кто клиент;
- проблемы;
- боли;
- страхи;
- желания;
- мотивацию;
- возражения;
- триггеры;
- ожидания;
- критерии выбора;
- словарь клиента;
- эмоциональные формулировки;
- основные вопросы.

Именно эти документы станут основой для поиска тем.

---

## ЭТАП 6. Формирование тем

Использовать следующую последовательность:

```
Услуга
   ↓
Портрет клиента
   ↓
Боли
   ↓
Возражения
   ↓
Желания
   ↓
Интересы
   ↓
Темы
```

Для каждого пула услуг сформировать максимально широкий список тем.

После этого выбрать лучшие.

И только потом распределить темы между аккаунтами.

---

## ЭТАП 7. Создание системы Claude

Создать единый главный репозиторий проекта.

В него добавить:

- методику Яны Ащиной;
- правила медицинского копирайтинга;
- принципы Максима Ильяхова;
- общие инструкции;
- универсальные промпты;
- шаблоны документов.

После этого создать отдельный проект Claude для каждого пула услуг.

Например:

- SMAS
- Лазерная эпиляция
- RF-лифтинг
- Инъекционная косметология
- Лечение акне
- Трихология
- и т.д.

Внутри каждого проекта разместить:

- документ по услуге;
- портреты ЦА;
- шаблон поиска тем;
- список тем;
- шаблон сценария;
- методику Яны Ащиной;
- примеры удачных сценариев;
- правила копирайтинга.

В результате каждый проект становится экспертом только по своей услуге.

---

## ЭТАП 8. Написание сценариев

Работать пакетами.

Например:

```
все сценарии по SMAS
   ↓
все сценарии по лазерной эпиляции
   ↓
все сценарии по RF
   ↓
и т.д.
```

Claude уже знает:

- услугу;
- аудиторию;
- боли;
- структуру;
- стиль;
- методику.

Поэтому остается только выбрать тему.

**Результат:**

45 полностью готовых сценариев.

---

## ЭТАП 9. Создание системы флаеров

На основе уже обученных проектов создать написание флаеров.

Ничего заново обучать не нужно.

Использовать:

- документ по услуге;
- портрет клиента;
- методику Яны;
- правила копирайтинга.

Подготовить:

- все флаеры филиала на Тренева;
- все флаеры филиала на Гоголя.
