# Контент-система клиники INNOVACIA

Переиспользуемая система для быстрого и качественного производства маркетинговых
материалов клиники по каждой услуге. В репозитории хранится **система**, а не её
продукты. Подробнее — в [«План действий.md»](План%20действий.md).

Направления: **косметология** и **лечение**.

## Структура репозитория

| Папка | Что внутри | Этап плана |
|-------|-----------|------------|
| `Этап 1: Охват-системы/` | Пулы услуг, список аппаратов, деление на направления | Этап 1 |
| `Этап 2: Методика-Ащиной/` | Методика Яны Ащиной: косметология и лечение | Этап 2 |
| `Этап 3: Правила-копирайтинга/` | Мед. копирайтинг, принципы Ильяхова, тон и стиль | Этап 3 |
| `Этап 4: Шаблоны/` | Идеальные шаблоны (документ, сценарий, ЦА, темы, флаер, ТЗ) | Этап 4 |
| `Этап 5-8: Услуги/` | По каждой услуге: документ, портрет ЦА, поиск тем, сырьё, эталонные сценарии | Этапы 5–8 |
