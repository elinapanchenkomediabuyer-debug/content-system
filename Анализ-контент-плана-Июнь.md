# Анализ контент-плана INNOVACIA — Июнь

Разбор месячного контент-плана (сторис + рилс + карусели) по услугам.
Источник: загруженный документ контент-плана на 1 июня – 5 июля.
**В подсчёты включены только даты июня (1–30 июня).** Даты 1–5 июля вынесены за скобки.

## Как считалось

- **Гранулярность — услуга.** Каждая строка = отдельная услуга/метод.
- **Сторис** = сюжетные блоки в ежедневном сторителлинге (проблема → решение → результат → CTA).
  Для **3D-диагностики** дополнительно учтены промо-слайды «Узнайте свой план красоты /
  дарим 3D-диагностику» (1, 4, 11 июня) — это регулярный лид-оффер клиники.
- **Посты** = карусели для ленты (формат «МАКЕТ АКЦИЯ → Карусель»).
- **Рилс** = рилс для ленты (формат «МАКЕТ АКЦИЯ → Название рилс»).
- Репосты уже опубликованных рилс/каруселей внутри сторис засчитаны как **сторис**
  (там они и выходят в июне), повторно в «Посты/Рилс» не считались.
- **12 рилс июня — образовательные/имиджевые, без привязки к конкретной услуге**
  (см. примечание внизу) — в счётчики услуг не попадают.

Итого за июнь: **47** сюжетных блоков в сторис, **6** каруселей (7 упоминаний услуг),
**20** рилс (8 с конкретной услугой + 12 образовательных).
